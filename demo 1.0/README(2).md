# AI杭州文旅策展人

本项目是浙江树人学院《人工智能》课程大作业。

## 项目简介

AI杭州文旅策展人是一个基于大模型的杭州个体化文旅定制系统。系统以智慧文旅为主题，base 杭州，面向大学生、本地居民与外地游客，根据用户的审美偏好、出行心情、预算和时间，生成个性化路线、杭州文化问答、Citywalk 智能规划和文旅灵感卡片。

系统强调杭州本地文化 IP 与现代审美表达，将西湖、良渚玉琮、京杭大运河、灵隐飞来峰、龙井茶园、钱塘江潮、亚运莲花等文旅资源转化为结构化、可交互、可展示的 AI 文旅策展体验。

## 核心功能

1. AI 个性化杭州旅行路线生成
   - 根据出行时长、出行人群、兴趣偏好、预算、旅行风格和出发区域生成杭州旅行路线。

2. 杭州文化 AI 问答助手
   - 回答用户关于杭州历史文化、景点、路线和旅行建议的问题。

3. 杭州 Citywalk 智能规划
   - 根据起点、步行时长、路线风格和预算生成适合大学生的杭州步行路线。

4. 文旅灵感卡片生成
   - 根据西湖、良渚、运河、灵隐、龙井茶、南宋街区等主题生成文旅卡片。

## 技术栈

- 前端：HTML5、CSS3、JavaScript
- 后端：Node.js、Express
- AI 能力：大模型 API 文本生成与问答
- 视觉风格：江南现代文旅风、玻璃拟态、低饱和色彩、ins 风城市策展页面

## 运行方式

### 1. 安装依赖

```bash
npm install
```

### 2. 启动项目

```bash
npm start
```

启动后访问：

```text
http://localhost:3000
```

## 大模型 API 配置

项目支持 OpenAI 兼容格式的大模型接口。复制 `.env.example` 为 `.env` 后填写：

```text
AI_API_KEY=你的API密钥
AI_API_URL=https://api.deepseek.com/chat/completions
AI_MODEL=deepseek-chat
PORT=3000
```

如果没有填写 API Key，系统会自动使用内置本地演示模型，方便课堂展示和离线运行。
