Core Functions

四个模块，完成一次杭州文旅策展。

01

AI 个性化路线

根据时长、人群、兴趣、预算、风格和出发区域生成杭州旅行路线。

02

杭州文化问答

回答杭州历史文化、景点背景、路线建议与旅行问题。

03

Citywalk 智能规划

围绕起点、步行时长、路线风格和预算生成大学生友好路线。

04

文旅灵感卡片

把西湖、良渚、运河、灵隐、龙井茶等文化资源转成可展示卡片。

About Project

面向课堂展示，也方便后续继续加审美设计。

当前版本采用 HTML5、CSS3、JavaScript、Node.js 与 Express，支持 OpenAI 兼容格式的大模型接口；没有配置 API Key 时，也可以直接运行本地演示。
